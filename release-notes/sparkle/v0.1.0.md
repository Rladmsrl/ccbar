## CCBar 0.1.0 — 首个公开发布

Claude Code 的原生 macOS 菜单栏 app。从本地 `~/.claude/` 读会话数据,把它变成随手一瞥就能看到的信息。

### 主要功能

- 菜单栏标签:5h 用量窗口剩余 / 百分比,可拖拽配置,与 statusLine 双向桥接
- 实时会话 HUD:谁在跑、停在哪、是否等待权限审批,行内 Stop / Respawn / Focus
- 可拖浮窗:一条分段色带映射所有会话,悬停看单会话预览 + 事件时间线
- 权限审批气泡:Claude 等待批准时弹出,Allow / Deny / Always 同步回 chat
- Token / Cost 统计:跨 session 去重,避免 cost 翻倍
- API / 配置切换台:一键切换 provider,管理配置 profile

需要 macOS 14+。内置 Sparkle 自动更新。
